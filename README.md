# Weather App

This is a simple weather app that fetches weather data from the OpenWeatherMap API.

## Requirements

- Node.js
- NPM

## Installation

1. Clone the repo:

git clone https://github.com/Muhammad-Rebaal/Weather-App.git


2. Install the dependencies:

npm install

1. Run the app:

npm start
